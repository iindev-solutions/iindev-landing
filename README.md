# iindev.

AI automation, Telegram bots, Bitrix, TMA, web platforms.

- **AI automation & agents** — автоматизация процессов, AI-сценарии, ассистенты, CRM и API-интеграции
- **Telegram bots & Mini Apps** — боты продаж, поддержки, внутренних процессов, TMA с полноценной логикой
- **Bitrix & corporate sites** — лендинги, корпсайты, каталоги, управляемая CMS и админские сценарии
